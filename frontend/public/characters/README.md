# Φωτογραφίες χαρακτήρων

Οι φωτό των χαρακτήρων δηλώνονται ρητά στο `app/page.tsx`, στο πεδίο `img` κάθε
χαρακτήρα (στον πίνακα `CHARACTERS`). Π.χ.:

```ts
{ id: "alekos", ..., img: "/images/alekos.jpg" }
```

**Για να βάλεις/αλλάξεις φωτό:**
1. Ρίξε το αρχείο μέσα στο `public/` (π.χ. `public/images/sakis.jpg`).
2. Όρισε το `img` του χαρακτήρα στο path αυτό (π.χ. `/images/sakis.jpg`).

**Προδιαγραφές:** τετράγωνη αναλογία (π.χ. 256×256 ή 512×512), `.jpg`/`.png`/`.webp`.

Αν το `img` λείπει ή το αρχείο δεν φορτώσει, εμφανίζεται αυτόματα ένα premium
monogram (Σ / Α) με κόκκινο gradient — δεν σπάει τίποτα.
